2024 Update

The BlockchainUniversity team is currently in the process of building the first, one of a kind decentralized operating system for compatible smartphones that can be jailbroken. As this is still the beginning of our journey, we are aiming to release a early, baseline version of the smartphone OS by Q2 in 2025. To stay updated on more regarding this next step into the right direction of Web 3.0 and those interested, please visit our GitHub repository for this project right here: https://github.com/blockchainuniversityedu/custom-rom
Once an early version releases, you will have to come to this page to download the built version of the ROM which will only be compatible with Android for the time being. If you're a developer interested in contributing to the project, you may contact us via Discord through this link: https://discord.gg/79EnNntzB7 or formally write us an email: blockchaintmu@gmail.com.
